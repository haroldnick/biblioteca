# Documentación de biblioteca

Esta es una documentación de prueba creada dentro del curso de Django utilizando la libreria
MkDocs para publicarlo en **ReadTheDocs**

## Requerimientos

* Clona el siguiente repositorio [Github](https://github.com/developerpe/biblioteca)
* `mkdocs serve` - Start the live-reloading docs server.
* `mkdocs build` - Build the documentation site.
* `mkdocs -h` - Print help message and exit.
